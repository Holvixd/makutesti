Who?

Hi I'm Vilho Stenman a 22 year old student living in Tampere Finland. I started stydying Business Information Technologies in Tampere University of Applied Sciences in 2015 'cause computers, games and other IT-stuff have always been close to my heart. Since when I started my studies we've come a long way and I should be graduating in about one year from now. Programming has become my "number one" thing and I'm constantly working on some of my own projects and of course on some mandatory school projects aswell. Working in the IT branch is like a dream come through for me since you can always learn new things and there is never a day you wouldn't have to solve a problem.

Considering how fast the IT-sector is evolving; new frameworks, new devices, new platforms, new languages etc. coming out each year or even every month, I've never been slowed down by any new technology replacing our old comfortable stuff that we've used to work with. With me it's more like I'm always excited and passionate for new things coming out and it's never been a problem for me to learn something new.

Some projects worth mentioning

WorkTimeTracker

Solo project

WorkTimetracker is an app made to track your working hours easily and to view them in a simple calendar. Both backend and frontend was made with java. The backend is no longer running on a server so the app doesn't function properly but the source code can be found from Github.

Online shop project

Team project

We made a simple online shop as a small group of 3 people. React.js was used for frontend and Spring Boot for backend. The server where the shop was running is no longer available but the source code can be viewed from Github.

Nutrirun

Team project

Nutrirun is a 2D platformer/runner learning game aimed for vocational school students. Our client in the project was UKK Institute and the game was made with libGDX. The game is downloadable from Google Play and the source code can be found from Github.

Postage stamp project

Team project

Postage stamp project was a school project where we used Finnish stamp opendata from avoindata.fi to create a webapp. Frontend was made with react.js and backend with java. The webapp is no longer running on a server but the source code can be viewed from Github.
